Aap apne Astro project ko [Render](https://render.com/) par deploy kar sakte hain, jo ek service hai jismein free TLS certificates, global CDN, DDoS protection, private networks, aur Git se auto deploys hain.

## Deploy kaise karein

<Steps>
1. Ek [render.com account](https://dashboard.render.com/) banayein aur sign in karein

2. Apne dashboard se **New +** button par click karein aur **Static Site** select karein

3. Apne [GitHub](https://github.com/) ya [GitLab](https://about.gitlab.com/) repository ko connect karein ya phir ek public repository ka public URL enter karein

4. Apni website ka naam dein, branch select karein aur build command aur publish directory specify karein
   - **build command:** `npm run build`
   - **publish directory:** `dist`, static sites ke liye; `dist/client` agar aap `server` ya `hybrid` server-rendering modes ka use kar rahe hain.
   - **Environment variables (advanced)**: By default, Render Node.js 14.17.0 use karta hai, lekin Astro [ek higher version ki zaroorat hoti hai](/en/install-and-setup/#prerequisites). Ek environment variable add karein **Variable key** ke saath `NODE_VERSION` aur **Value** `v18.17.1` ya `v20.3.0` ya higher, taaki Render ek compatible Node.js version use kare. Ya phir apne project mein ek [`.node-version`](https://render.com/docs/node-version) ya [`.nvmrc`](https://render.com/docs/node-version) file add karein Node.js version specify karne ke liye.

5. **Create Static Site** button par click karein
</Steps>
